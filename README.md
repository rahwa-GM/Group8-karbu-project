# Karbu Project - Procurement & Sales Forms

This project is a **web-based form system** for managing procurement and sales entries. It includes responsive forms for entering supplier, product, and customer information, along with payment details.  

## Features

- **Procurement Form**  
  - Supplier details (name, contact)  
  - Purchase details (date, invoice, description, product info)  
  - Payment info (method, status, delivery date)  
  - Auto-calculated total amounts  

- **Sales Form**  
  - Customer information (name, phone, email, type)  
  - Product details (name, category, quantity, unit price, discounts)  
  - Auto-calculated subtotal, tax/VAT, and total  
  - Payment info (method, amount paid, balance)  
  - Additional notes  

- **Modern UI**  
  - Clean design using **CSS3 features** (Flexbox, gradients, transitions)  
  - Responsive layout for desktop and mobile  
  - Styled radio buttons, buttons, and inputs  
  - Background image with blur effect  

## Technologies Used

- HTML5  
- CSS3 (Flexbox, gradients, transitions)  
- Google Fonts (Poppins)  
- Font Awesome icons
